## 📱 About App

RecipeFinder — удобное приложение для поиска и изучения рецептов со всего мира. Оно позволяет находить блюда по названию или кухне, показывая инструкцию по приготовлению, время приготовления. Приложение одинаково полезно как для домохозяек, которые хотят быстро приготовить вкусный ужин, так и для профессиональных шеф-поваров, ищущих вдохновение в кулинарных традициях разных стран.


## 🚀 Features

### 🛡️ Onboarding
Приложение имеет вводную часть - онбординг, которая состоит из 4 слайдов которые посвящены возможностям приложения. После нажатия на кнопку Start вы попадаете на начальный экран приложения.


### 🧑‍🍳 HomeView
На основном экране:
    Рецепты раздела "Популярное"
    Категории для загрузки рецептов таких блюд как : десерты, салаты, супы и пр.
    Просмотренные рецепты 
    Кухни мира - подборка кухонь, по которым можно найти рецепты и просмотреть их детальную информацию
    В нижней части экрана реализован TabBar для удобной навигации
    
    
### 🧭 TabBar
TabBar включает в себя возможность легкой навигации по экранам: 
    HomeView - главный экран приложения
    FavoritesView - экран, который ханит избраные рецепты, отмеченный пользователем
    NotificationView - экран с напоминалками
    ProfileView - экран пользовательского профиля с фоторграфией и собственными рецепами
    CreateView - центральная кнопка TabBar, для создания своего собственного рецепта
    

### 🔎 SearchView
Экран, на который попадаешь сразу после нажатия на поисковую строку на главном экране.
    Ищет рецепты по ключевому слову и отображает их на экране
